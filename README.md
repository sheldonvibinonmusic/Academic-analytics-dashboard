# 📊 Academic Analytics Dashboard

## 📌 Overview

This project analyzes student academic performance using **Python** and **Pandas**. It reads student data from a CSV file and generates meaningful insights such as subject statistics, student rankings, attendance analysis, study hours analysis, and grouped summaries.

---
## ✨ Features

- 📋 Dataset overview
- 📚 Subject-wise mean, maximum, and minimum marks
- 🧮 Automatic total and percentage calculation
- 🏆 Highest and lowest performer identification
- 🎯 Student performance categorization
- 📅 Attendance analysis
- ⏰ Study hours analysis
- 👨‍🎓 Individual student strengths and weaknesses
- 🔝 Top 10 and Bottom 10 student rankings
- 📊 Attendance category summary using GroupBy
